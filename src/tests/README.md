# Tests / Tests

## English
This directory contains the test suite for the application. We use **Vitest** as the test runner and **React Testing Library** for component testing.

**Structure:**
- `setup.ts`: Global test setup configuration (e.g., extending Vitest matchers).
- `*.test.tsx`: Test files. Each major feature or component should have a corresponding test file.

**Running Tests:**
Since this is a standard architecture, you would typically run:
```bash
npx vitest run
```
