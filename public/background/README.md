# Background Images / Images d'arrière-plan

## English
This directory is intended to store background images and other static assets that should be served publicly.
For example, you might place a `hero-background.jpg` here and reference it in your code as `/background/hero-background.jpg`.

**Usage:**
```tsx
<div style={{ backgroundImage: "url('/background/your-image.jpg')" }}>
  ...
</div>
```

## Français
Ce dossier est destiné à stocker les images d'arrière-plan et autres ressources statiques qui doivent être servies publiquement.
Par exemple, vous pouvez placer un fichier `hero-background.jpg` ici et y faire référence dans votre code via `/background/hero-background.jpg`.

**Utilisation :**
```tsx
<div style={{ backgroundImage: "url('/background/votre-image.jpg')" }}>
  ...
</div>
```
